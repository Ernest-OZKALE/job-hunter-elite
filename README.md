Bienvenue dans le dépôt **Job Hunter Elite**, une solution "Software Architect" conçue pour transformer la gestion des candidatures en une expérience de niveau professionnel supérieur.

---

## 🏛️ Vision Architecturale

Ce projet n'est pas une simple application CRUD. Il a été conçu comme un **écosystème modulaire** répondant aux règles de l'art du développement moderne :

-   **Architecture Monorepo** : Séparation stricte entre le code source (`apps/`), la documentation technique (`docs/`), et l'infrastructure cloud (`infrastructure/`).
-   **Localisation Totale** : Documentation 100% en Français pour une clarté maximale en environnement professionnel francophone.
-   **Hybride Cloud/Local** : Intégration de services managés (Supabase, Vercel) avec des capacités d'automatisation HomeLab (n8n, scraping).
-   **IA-First** : Intégration native de l'IA (Gemini 1.5 Flash) pour le scoring et l'assistance à la rédaction.

## 🚀 Démarrage Rapide

```bash
# Installation des dépendances à la racine
npm install

# Lancer l'application client en développement
npm run client:dev
```

## 🛡️ Audit de Qualité

-   **Zéro Secrets** : Aucune clé API n'est hardcodée (utilisation stricte des `.env`).

-   **Performance** : Optimisé via Vite 7 et Tailwind v4 pour des temps de chargement records.
